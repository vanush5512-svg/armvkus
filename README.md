## БЫСТРЫЙ СТАРТ (бесплатно)

### 1. Настрой Telegram бота (5 минут)
1. Напиши `@BotFather` в Telegram
2. Отправь `/newbot`
3. Придумай имя боту (например `armvkus_orders_bot`)
4. Скопируй токен вида `123456789:ABCdefGHI...`
5. Напиши боту любое сообщение
6. Открой в браузере: `https://api.telegram.org/bot[ТОКЕН]/getUpdates`
7. Найди `"chat":{"id":123456789` — это твой CHAT_ID

### 2. Вставь настройки в сайт
Открой `index.html`, найди строки:
```javascript
var TG_BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА';
var TG_CHAT_ID = 'ВАШ_CHAT_ID';
var ORDER_PASSWORD = '1111'; // Пароль для оформления заказа
```

Замени на свои значения. Можешь сменить пароль.

## ХОСТИНГ (выбери вариант)

### Вариант А: GitHub Pages (бесплатно, но .github.io домен)
1. Создай аккаунт на github.com
2. Создай новый репозиторий `armvkus`
3. Загрузи все файлы (index.html, data.js, ararat.png)
4. Настройки → Pages → Включи GitHub Pages
5. Сайт будет на `https://твойник.github.io/armvkus`

### Вариант Б: Timeweb (домен .ru ~200₽/год)
1. Зарегистрируйся на timeweb.cloud
2. Купи домен `.ru` (~200₽)
3. В разделе "Сайты" создай новый
4. Загрузи файлы через FTP или файловый менеджер
5. Готово! Сайт на твоём домене

### Вариант В: Netlify (бесплатно, можно свой домен)
1. Зарегистрируйся на netlify.com (через GitHub)
2. Перетащи папку с файлами на страницу
3. Получишь ссылку типа `armvkus-abc123.netlify.app`
4. Можно купить домен .ru и привязать
